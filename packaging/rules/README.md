# Bundled Hashcat Rule Files

**OneRuleToRuleThemAll.rule** — created by NotSoSecure; distributed under an
MIT-compatible license; powers Stage 7 of the attack cascade with a large,
curated set of transformation rules optimised for real-world passwords.

**best64.rule** — part of the hashcat project (hashcat/hashcat); distributed
under the MIT license; powers Stage 6 with 64 high-yield, broadly applicable
password transformation rules.

**dive.rule** — part of the hashcat project (hashcat/hashcat); distributed
under the MIT license; provides an exhaustive dive through common password
mutation patterns and is used as a supplementary rule set in later cascade
stages.
